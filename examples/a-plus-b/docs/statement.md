# A+B Problem

## 题目描述

给定两个整数 $a$ 和 $b$，请你输出它们的和。

虽然这是一道经典入门题，但本题中的整数范围较大，请注意使用足够大的整数类型。

## 输入格式

输入仅一行，包含两个整数 $a$ 和 $b$。

## 输出格式

输出一行一个整数，表示 $a+b$ 的值。

## 数据范围

- $-10^{18} \le a,b \le 10^{18}$

## 样例

```input1
1 2
```

```output1
3
```

```input2
-1000000000000000000 999999999999999999
```

```output2
-1
```

## 说明

第二组样例说明输入可能为负数，并且数值接近边界。
