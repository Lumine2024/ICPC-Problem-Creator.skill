# A+B Problem Tutorial

## 题意分析

题目要求输出两个整数 $a$ 和 $b$ 的和，没有额外操作，也没有多组数据。

真正需要注意的点只有一个：数据范围达到 $10^{18}$ 级别，因此在 `C++` 中必须使用 `long long`，不能使用 `int`。

## 做法

直接读入两个整数，然后输出 `a + b`。

## 正确性证明

设读入的两个整数分别为 $a$ 和 $b$。

题目要求输出的结果正是它们的和，即 $a+b$。算法在读入后直接计算并输出 `a + b`，因此输出值与题目定义完全一致。

所以该算法对所有合法输入都正确。

## 复杂度分析

- 时间复杂度：`O(1)`
- 空间复杂度：`O(1)`

## 易错点

1. 使用 `int` 存储输入，可能在边界数据上溢出。
2. 忘记处理负数，写成诸如 `abs(a) + abs(b)` 之类的错误式子。
