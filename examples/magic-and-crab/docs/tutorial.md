# 魔法与螃蟹 题解

## 题意概述

隐藏整数 `x` 满足 `2 <= x <= 10^18`。每次可以选择 `2 <= y <= 100` 询问，得到 `gcd(x, y)`。最多询问 `20` 次，最后输出一个 `2 <= z <= 100`，要求 `gcd(x, z) = 1`。

## 正解思路

依次询问若干个小质数。如果某个质数 `p` 的返回值是 `1`，就说明 `p` 与 `x` 互质，直接输出 `p` 即可。

为什么一定能在前 `20` 个质数里找到这样的 `p`？

如果前 `20` 个质数都不与 `x` 互质，那么 `x` 必须同时被它们整除，也就是它们乘积的倍数。可这 `20` 个质数的乘积已经远大于 `10^18`，与题目的上界矛盾。

因此，前 `20` 个质数中一定至少有一个与 `x` 互质。

## 正确性说明

当查询 `p` 后得到的返回值为 `1` 时，定义上就有 `gcd(x, p) = 1`。此时输出 `p` 一定合法。

又因为前 `20` 个质数不可能全部整除 `x`，所以算法一定会在不超过 `20` 次查询内找到答案。

## 复杂度分析

- 询问次数：至多 `20`
- 时间复杂度：`O(20 log 100)`
- 空间复杂度：`O(1)`
